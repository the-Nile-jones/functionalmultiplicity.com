The Handbook teaches You how to find Your atoms and build with them. Tools is where You learn how to set up the infrastructure that makes all of it persist — across switches, across sessions, across time.

AI that knows Your System. Notes that survive a front. A library that grows without You having to rebuild context every time.

What's coming

The Tools page will cover setup in tiers — starting at zero cost, expanding as far as You want to go.

Tier 0 Free

One AI instruction. Copy it into any AI assistant and You're set up. No accounts beyond the AI itself, no configuration, no maintenance. This is the minimum viable starting point — and it works.

Tier 1 Free

Notion + an MCP connection. Your atoms live in a document Your AI can read directly. When someone new fronts, the AI reads the library and picks up where the last Member left off. Persistence across switches without anyone having to re-explain.

Tier 2 Free or low-cost

Task management + a second MCP connection. What needs doing, who committed to what, what's in progress — the AI knows. No more rebuilding context from scratch after a switch.

Tier 3 Varies

The full Exocortex. Multiple MCP servers, skill files, version-controlled AI instructions, biometric fronting detection, and a library of 400+ atoms. Built and refined over four years. This is the Nile System's actual setup — documented and made replicable.

You don't need Tier 3 to get value. Tier 0 works. Start there and expand when You're ready.
